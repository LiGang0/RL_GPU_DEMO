# ELEN 6885 Reinforcement Learning Workshop

## Test Platform
- Google Compute Engine
- 2 CPU with 6GB RAM
- 1 NVIDIA K80 GPU
- HTTP Opened

## Setup
Install CUDA, NVIDIA Driver, CUDA Toolkit, Tensorflow, keras, Jupyter Notebook
```
$ bash setup.sh
```
Run a simple Jupyter Notebook Host on http://0.0.0.0:80/
```
$ bash run.sh
```
